# Changelog

## [0.1.0] - 2026-03-15

### Added

- **Workspace Explorer**: Project tree view grouped by folder, with project type icons, targets, and dependencies
- **Target Runner**: Run and Run Verbose targets from the sidebar, command palette, or code lens
- **Code Lens**: Inline "Run | Run Verbose" buttons above target definitions in `pubspec.yaml` files
- **Generator UI**: Webview form for `fx generate` with generator selection, dry-run preview, and file generation
- **Graph Visualization**: Interactive dependency graph webview with search filter, click-to-highlight, and VS Code theme support
- **Task Status Panel**: Task execution history with status icons, duration tracking, and re-run support
- **Settings**: Configurable fx binary path, concurrency, auto-refresh, and code lens toggle
- **Keyboard Shortcuts**: `Ctrl+Shift+R` (Run), `Ctrl+Shift+G` (Graph), `Ctrl+Shift+N` (Generate)
- **Status Bar**: Shows workspace project count
